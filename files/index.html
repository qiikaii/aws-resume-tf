Automated Patch Monitoring System

Cloud-Native Web Crawling & Notification Workflow

A highly scalable automated monitoring system built on AWS to track and notify the latest patches and hotfixes for 50+ products. The solution uses Amazon EventBridge to trigger scheduled executions of AWS Step Functions, which orchestrate Lambda functions running Selenium web crawlers. Results are stored in DynamoDB, and notifications are distributed via SNS and SES. The serverless design allows the system to easily scale to support hundreds of products while reducing manual effort and improving efficiency in patching and securing customer websites.

Note: This is an internal company project - code and documentation are confidential.
